# Para + Safe

## Simple example app

This is simple example app that integrate Para to login and Safe to execute transactions. 


### Install the packages
```
  npm install 
```

### Run in development 
```
  npm run dev
```

Check the demo: [PARA+SAFE](https://capsule-safe.vercel.app/)
